Optional recoveryLevel

recoveryLevel: DeletionRecoveryLevel
  • Defined in keysModels.ts:69
member

{DeletionRecoveryLevel} [recoveryLevel] Reflects the deletion recovery level currently in effect for keys in the current vault. If it contains 'Purgeable' the key can be permanently deleted by a privileged user; otherwise, only the system can purge the key, at the end of the retention interval. Possible values include: 'Purgeable', 'Recoverable+Purgeable', 'Recoverable', 'Recoverable+ProtectedSubscription' NOTE: This property will not be serialized. It can only be populated by the server.
